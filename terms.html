利用規約

この利用規約（以下、「本規約」といいます）は、株式会社Gradus（以下、「当社」といいます）が提供するアプリケーション及び関連ソフトウェア（以下併せて、「本アプリ」といいます）の利用に関する条件を、本アプリを利用する利用者（以下、「利用者」といいます）と当社の間で定めるものです。

第1条 効力の発生

  1. 利用者は、本規約の内容に承諾したうえで、本アプリを利用するものとします。また、本規約は利用者が本アプリを利用した日をもってその効力を発するものとします。
  2. 利用者は、本アプリ内において個別に定める規約及び本アプリ内で記載する注意事項等（以下、「個別規約」といいます)が定められているときは、本規約及び個別規約に基づき本アプリを利用するものとします。
  3. 利用者が未成年である場合は、親権者等法定代理人の同意を得たうえで本アプリを利用するものとします。

第2条 本アプリの利用

  1. 本アプリを使用するために必要な携帯端末、通信機器、通信手段等は、利用者の費用と責任で備えるものとします。
  2. 利用者は、本アプリの利用にあたっては、本規約及び個別規約のほか、著作権法等の法令を遵守するものとします。

第3条 会員

  1. 会員とは、本アプリへの入会を申込み、当社が承認した利用者をいいます。
  2. 入会の申込みをする利用者は、当社が定める手続きに従い、会員登録を行うことが必要です。
  3. 当社は、申込者が次のいずれかに該当する場合は、その者の会員契約の申込を承諾しないことがあります。
    1. 申込者が実在しない、もしくは実在しないおそれがあると判断した場合
    2. 申込の時点で、会員規約の違反等により、IDの一時停止、強制退会処分もしくは会員契約申込の不承諾を現に受け、又は過去に受けた事がある場合
    3. 申込の際の申告事項に、虚偽、誤記、又は記入漏れがあった場合
    4. 過去に支払いを怠ったことがある場合
    5. 申込の際に決済手段として当該申込者が届け出たクレジットカードがクレジットカード会社により無効扱いとされていること、又は当社の指定する決済関係先が当該申込者との契約の締結を拒否した場合
    6. その他、当社が会員として不適当と判断した場合
  4. 会員は、会員たる地位及び会員として有する権利を第三者に譲渡又は質入れその他第三者の権利を設定することはできないものとします。

第4条 ID・パスワードの管理

  1. 会員は、会員登録等の際に設定した会員ID及びパスワードを厳重に管理・保管する責任を負うものとします。
  2. 会員は、会員ID及びパスワードを第三者に譲渡、売買、承継、貸与、開示又は漏洩してはならないものとします。
  3. 会員は、会員ID又はパスワードの管理不十分、使用上の過誤・不手際、第三者の無断使用等に起因する損害につき自ら責任を負うものとし、当社は一切の責任を負いません。
  4. 当社は会員の会員ID及びパスワードを使用した本アプリの利用は、すべて会員ご自身からの利用とみなします。
  5. 当社に対する支払いにチャージバックが生じた場合、当社では、その支払いが利用者の同意なく実行されたものであるとみなし、かつ利用者の個人情報が侵害されたものと受け止め、アカウントを停止するものとします。

第5条 有料アイテム

  1. 利用者は、本アプリで有料アイテムが購入できる場合は、当社が定める方法で購入することができます。
  2. 有料アイテムは、購入又は配布された端末でのみ利用することができ、本アプリのアンインストール及び携帯端末等の機種変更を行った場合、アイテムは引き継がれません。

第6条 ポイント

  1. 利用者は、本アプリにおいてポイント取得が可能の場合は、本アプリへの会員登録を行う必要があります。
  2. ポイントは、本アプリにおいてコンテンツの利用又は購入に必要な支払いでのみ利用可能とし、その他の使途や目的で利用してはならないものとします。
  3. 会員は、ポイントを本アプリ内でのみ消費することができます。
  4. ポイントの有効期限は、会員が獲得した月を1ヵ月目として起算して6ヵ月目末日とします。当該有効期限が経過した未消費のポイントは自動的に消滅します。

第7条 スポンサーサイト経由によるポイント付与

  1. 会員は、ポイント付与の対象となるスポンサーサイトの条件を満たした場合、ポイントを取得することができます。
  2. ポイントの提供内容及び提供期間は、スポンサーサイト、登録条件ごとに当社が定めるものとします。
  3. ポイントを付与するか否かは、スポンサーサイト及び当社の判断によるものとします。
  4. スポンサーサイト側で発生した問題について、当社は一切その責任を負わないものとします。
  5. スポンサーサイトがなんらかの理由で、会員の行為を承認しない場合、ポイントは加算されません。その際、当社はポイントが加算されない理由を会員に説明する義務を負わないものとします。

第8条 有料アイテム及びポイントの利用

  1. 当社は、当社の判断により有料アイテム及びポイントを利用者に無料で配布することができます。
  2. 利用者は、有料アイテム及びポイントを、他の利用者を含む第三者に利用させる他、貸与、譲渡、売買、質入等をすることはできません。
  3. いかなる事情があろうと、有料アイテム及びポイント購入後の取り消し、返金は一切できません。
  4. 有料アイテム及びポイントは将来にわたって本アプリや特典を利用できる権利を保証するものではなく、本アプリや特典の内容の変更、及びシステムの変更による変更や廃止その他いかなる場合であっても、利用者が所持している有料アイテム及びポイントの利用について当社は保証しないものとします。
  5. 保守作業、システム復旧、停電や天災などの不可抗力、又はその他のやむをえない理由により、誤った内容のポイントが会員に付与された場合、当社は、正確な内容に修正変更ができるものとします。

第9条 有料アイテム及びポイントの無効・消滅・取り消し

当社は有料アイテム及びポイントの取得又は利用に関して下記に該当する場合は予告なく有料アイテム及びポイント無効にします。また、有料アイテム及びポイントの利用・購入を制限する場合があります。無効になったポイントの返還・交換は行いません。

  1. 違法又は不正行為があった場合
  2. 本規約又は個別規約の違反があった場合
  3. アカウントが停止になった場合
  4. 当社が定めた方法以外で不正に有料アイテム及びポイントを取得した場合
  5. 保守作業、システム復旧、停電や天災などの不可抗力、エラー、バグ、不具合、セキュリティ上の欠陥又はその他のやむをえない理由により、誤った内容のポイントが会員に付与された場合
  6. 会員が本アプリから退会した場合
  7. その他当社が会員に付与された有料アイテム及びポイントを取り消すことが適当と判断した場合

第10条 禁止事項

  1. 当社は禁止事項に違反した利用者に対して、本アプリの利用停止、アカウント凍結、及び有料アイテム及びポイント無効化等を含むあらゆる措置又は手続を、事前の通知なく行う事ができるものとします。
  2. 利用者の行為が禁止事項に該当するか否かは、当社の判断によるものとします。
  3. 禁止行為に基づき当社又は第三者が損害を被った場合は、利用者はこれを賠償するものとします。

第15条 権利の消失

  1. 本アプリの退会は、対応端末を通じて、当社が定める手続を行うことが必要です。

第17条 紛争処理

  1. 本規約は日本法に準拠し、同法に基づいて解釈されるものとします。
